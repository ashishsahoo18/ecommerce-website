# 🛒 Block Rock - E-Commerce Website

A modern and responsive **E-Commerce Website** developed using **HTML, CSS, and JavaScript**.  
Block Rock was created to provide a complete online shopping experience with product browsing, search, filtering, cart management, and user authentication features.

## ✨ Features Implemented

### 🔐 User Authentication System
- Implemented user signup functionality
- Implemented user login functionality
- Added logout feature
- Managed user data using Local Storage
- Created user account handling system

### 🛍️ Shopping Features
- Developed product listing section
- Added product cards with images, price, ratings, and discounts
- Implemented Add to Cart functionality
- Added wishlist interface
- Added product search functionality
- Created category-based product browsing

### 🎯 Product Filtering System
- Implemented category filters
- Added brand filtering
- Added price range filtering
- Added rating-based filtering

### 🔥 Flash Sale Section
- Designed flash sale banner
- Added countdown timer
- Displayed discount offers

### 🎨 UI Design
- Created modern e-commerce interface
- Designed responsive layout
- Added navigation menu
- Created hero banner section
- Added category cards
- Designed product grid layout
- Added dark mode interface
